# **What is a Server?**
***Its called a server because it provides services to other computers and its users/clients***

- Servers are typically housed in data centers, which ensure continuous power and internet connectivity.
	**Example**: [Nginx](https://www.f5.com/products/nginx#products)

# **Server Hardware and Software**

- Server hardware refers to the physical components, while software is the code that runs on the hardware.
- Different servers are built for specific purposes, such as:
- **image storage (Storage focused hardware)**
- **complex calculations. (Computation focused hardware)**

# **Web Servers and Their Functions**

- A web server:
	- **stores** and 
	- **manages data**, 
	- **administers websites**, 
	- **ensures security**, and 
	- **handles email**.
- It responds to web requests from clients, facilitating the [[What is a Request Response Cycle|request-response cycle]], and can handle thousands of requests per second.
