
# Types of Web Hosting

- **Shared Hosting**: The most affordable option where multiple websites share the same server resources. Suitable for small websites with low traffic.
- **Virtual Private Server (VPS)**: Offers dedicated resources (CPU, memory, bandwidth) for each user, providing better performance than shared hosting but at a higher cost.

# Dedicated and Cloud Hosting

- **Dedicated Hosting**: A physical server dedicated solely to one user, providing complete control over resources. This option is generally more expensive.
- **Cloud Hosting**: 
	- Utilizes multiple physical and virtual servers to host a website, 
	- If a Physical or virtual server fails, The website will run on a diff server and stay online.
	- ensuring high availability and scalability. (No hardware limitations)
	
**Example:** when we transfer or download a file to a server or from a server 
- Users pay based on bandwidth usage, which can be cost-effective for growing websites (aws or azure)
