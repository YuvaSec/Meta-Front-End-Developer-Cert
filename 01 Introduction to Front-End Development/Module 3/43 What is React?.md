
- React focuses on components, making it easier to create functional user interfaces for web and mobile applications.
- It is an open-source library, maintained by a community of contributors, and is widely used for developing single-page applications.

# Component MODEL in React
 - Components are small pieces of user interface that can be reused throughout an application, improving efficiency and maintainability.
