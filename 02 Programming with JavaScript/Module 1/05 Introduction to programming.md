# Understanding Programming

- Programming involves giving computers a set of instructions using a programming language.
- A computer program is a series of instructions that tells a computer what to do under specific circumstances.

# Programming Languages
- Programming languages serve as a bridge for communication between 
	- #### humans and computers.
- High-level languages like :
	- JavaScript need to be interpreted into binary code for the computer's CPU to understand, 
	- while low-level languages are closer to machine language.
