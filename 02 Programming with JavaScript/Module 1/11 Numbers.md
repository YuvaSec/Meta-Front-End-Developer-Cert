# Understanding Number Data Type

- JavaScript supports both ==' integer '== and ==' decimal numbers '==, which are essential for coding.
- Examples:
	- 123 and 123.456.

# Basic Mathematical Operations
- Common operations include 
	- addition (+), subtraction (-), multiplication (*), and division (/).
- ==The modulus operator ***(%)*** is used to find the remainder of division==.

# Order of Operations
- Parentheses ==" () "== can change the order of calculations, 
	- 2*(4+8) = 24.
- Without parentheses, 
	- 2 * 4+8 = 16
