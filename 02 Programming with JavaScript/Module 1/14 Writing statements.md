# IF CONDITION
- Conditional statements allow developers to execute code based on whether a condition is true or false.
	![[Screenshot 2025-11-29 at 16.25.40.png]]
- The if statement checks a condition and executes a code block if the condition is true.



# ELSE CONDITION 
- The if-else statement handles two outcomes: 
	- if the condition is true, one block of code runs; if false, another block runs.![[Screenshot 2025-11-29 at 16.26.10.png]]
- An example is provided where a student's driving test score determines if they pass or fail based on a threshold.
	![[Screenshot 2025-11-29 at 16.29.32.png]]




# ELSE IF CONDITION
- For more complex scenarios, the else-if statement can be used to test multiple conditions.
	![[Screenshot 2025-11-29 at 16.26.54.png]]
- This allows for more nuanced decision-making in programming, accommodating various outcomes based on different criteria.
