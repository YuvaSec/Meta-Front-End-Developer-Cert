


# **Null Datatype**

- #important Null Datatypes Represents the intentional absence of any object value.
- Returned by some built-in JavaScript methods when a value cannot be found.



# **Undefined Datatype**

- #important Undefined datatype Indicates a variable that has been declared but not assigned a value.
	![[Screenshot 2025-11-30 at 01.46.41.png]]
- Functions return undefined by default unless a specific value is returned.
- #important This can create TEMPORAL DEAD ZONE (where a variable is declared but not assigned)

# **Empty Strings**

- A string that contains no characters, created using single or double quotes with no characters in between.
- Represents a valid string value but is empty.
