
# Component Hierarchy in React![[Componet hierarchy.mp4]]

- Every React application starts with a root component, known as the app component, which can have multiple child components.
- An example is provided with a shopping list application, where the app component includes a new item bar and a shopping list component, demonstrating how components can be reused.

# Breaking Down a Website into Components![[index 3.mp4]]

- A typical blog website structure is discussed, featuring an app component, a Navbar component, and a Page component.
- The Page component contains a main feature component and two instances of a small feature component, showcasing the reusability of code in React.
