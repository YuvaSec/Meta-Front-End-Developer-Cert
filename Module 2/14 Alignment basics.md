
# **Text Alignment**
```css
p {
	text-align: center;
}
```

Text alignment can be set to left, right, center and **==justify==**.
#### The ==**justify** alignment==:
spreads the text out so that every line of the text has the same width.



# **HTML Element Center Alignment**
```html
<div class="parent">
	<div class="child">
	</div>
</div>
```

```css
.parent {
	border: 4px solid red;
}

.child {
	width: 50%;
	padding: 20px;
	border: 4px solid green;
	margin: auto;
}
```
