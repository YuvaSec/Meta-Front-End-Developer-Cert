
# Block and Inline Elements
![[block and inline tags.mp4]]
- Block-level elements occupy the full width of their parent and start on a new line, stacking vertically (e.g., `<div>`, `<form>`, `<h1>`).
- Inline elements only take up the width of their content and do not start on a new line, allowing them to flow horizontally (e.g., `<span>`, `<a>`, `<img>`).

# Demonstration of Document Flow

- An example is provided using a `<div>` containing sentences and `<span>` elements, illustrating how changing a `<span>` to a `<div>` moves the text to a new line.
- The display property in CSS can change an element's behavior from block to inline and vice versa.

# CSS Display Property

- The display property can be set to control how elements are rendered, with options like `inline` and `block`.
- Understanding these properties is crucial for web developers to manage layout effectively.
